<h1 align="center"><strong>React E-Commerce — Ecommerce Web App built with React + Vite + Basic Backend </strong></h1>
<p align="center"> Aplicação web de ecommerce desenvolvida em <strong>React.js</strong> com <strong>Vite</strong>, arquitetura de componentes reutilizáveis, gerenciamento simples de estado e integração simulada com APIs. Criado como projeto de estudo para aprimorar habilidades em desenvolvimento front-end, interfaces dinâmicas e práticas modernas de código limpo.</p>
<br><br>
<p align="center"><em>Ecommerce web application built using <strong>React.js</strong> and <strong>Vite</strong>, featuring reusable components, state management, and simulated API integration.
Developed as a study project focused on modern front-end development and clean code practices.</em></p>

<h2 align="center"><strong>Sobre o Projeto / About the Project</strong></h2>
<p align="center"> O ReactCommerce simula um fluxo completo de um site de compras moderno: listagem de produtos, carrinho, checkout e histórico de pedidos. Todo o front-end foi desenvolvido com foco em componentização, reutilização, estados isolados e integração com uma API simulada. Além disso, foi implementado um sistema básico de backend para gerenciar o carrinho e as compras </p> <p align="center"> <em> This project simulates a full ecommerce flow: product listing, cart system, checkout and order history. The front-end is entirely built based on reusable components, state management and mock API interaction. Beyond that, a basic backend system was also implemented for managing the cart and product orders, with the use of JSON. </em> </p>
